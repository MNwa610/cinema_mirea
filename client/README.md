# Фронтенд для афиши кинотеатров

React приложение для просмотра кинотеатров, фильмов.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Структура проекта

- `src/App.jsx` - главный компонент с роутингом
- `src/components/` - переиспользуемые компоненты (Header, модальные окна)
- `src/pages/` - страницы приложения
- `src/styles/`- стили приложения

## API

Приложение использует бэкенд API на порту 5050. Убедитесь, что бэкенд запущен.

## Особенности

- Авторизация и регистрация пользователей
- Просмотр кинотеатров поблизости
- Выбор фильмов в кинотеатре
- Детальная информация о фильмах
- Места для подключения карт

## Цветовая схема

- Синий (#1e3a8a) - основной цвет
- Белый (#ffffff) - фон
- Черный (#000000) - текст
